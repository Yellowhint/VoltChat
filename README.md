# SpeakEase Chat

A modern, responsive chat interface with typing indicator and dark mode.

## Features

- User & bot chat bubbles
- Typing indicator for bot responses
- Enter key to send messages
- Light/Dark theme toggle
- Responsive design

## Usage

1. Open `index.html` in a browser.
2. Type a message and hit Enter or click Send.
3. Bot will respond automatically.
